1a. Serverseitige Datenspeicherung (Supabase) / Server-Side Data Storage (Supabase)

Die App übermittelt anonymisierte Auslastungsbeobachtungen an einen externen Server (Supabase, gehostet in der EU), um Prognosen für alle Nutzer zu erstellen. Folgende Daten werden dabei übertragen:
The app transmits anonymized occupancy observations to an external server (Supabase, hosted in the EU) to generate forecasts for all users. The following data is transmitted:

  • Studio-ID, Wochentag, Uhrzeit und Auslastungsprozent / Studio ID, weekday, hour and occupancy percentage
  • Ein zufällig generierter, anonymer Geräte-Hash (16 Zeichen) zur Deduplizierung / A randomly generated, anonymous device hash (16 characters) for deduplication
  • Datum der Beobachtung / Date of the observation

Es werden keine IP-Adressen, Geräte-IDs, Standortdaten oder sonstige personenbezogene Daten gespeichert oder übermittelt. Der Geräte-Hash ist nicht rückverfolgbar und dient ausschließlich der Vermeidung von Doppeleinträgen.
No IP addresses, device IDs, location data or other personal data is stored or transmitted. The device hash is not traceable and is used solely to prevent duplicate entries.

The app uses Firebase Analytics, an analytics service provided by Google LLC (1600 Amphitheatre Parkway, Mountain View, CA 94043, USA). Firebase Analytics collects pseudonymized usage data to improve app functionality. The following data categories may be processed:

3. Speicherfristen / Storage

Die an Supabase übermittelten anonymisierten Auslastungsdaten werden serverseitig zur Berechnung von Durchschnittswerten gespeichert. Da diese Daten keinen Personenbezug aufweisen, ist eine Löschung einzelner Nutzer-Beiträge nicht möglich. Firebase Analytics speichert Analysedaten standardmäßig für 14 Monate. Es werden keine personenbezogenen Daten gespeichert.
The anonymized occupancy data transmitted to Supabase is stored on the server to calculate averages. Since this data contains no personal reference, deletion of individual user contributions is not possible. Firebase Analytics retains analytics data for 14 months by default. No personal data is stored.

No personal data is shared with third parties. Data transmitted to Supabase is fully anonymized. Firebase Analytics and Google Firestore receive pseudonymized or non-personal data and process it as data processors pursuant to Art. 28 GDPR.
